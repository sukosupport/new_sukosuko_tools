FAQ - Frequently Asked Questions

About the App

The SHOWROOM Sukosuko Tools is an unofficial app. However, we are in collaboration with the official team, so there may be changes to the functionality from time to time. We appreciate your understanding.

User Information and Privacy

User information and personal data, such as login ID, password, cookies, and user ID, are only used within the app and for communication via the SHOWROOM official API. We do not manage or use this information outside of these purposes, so please rest assured.

For Those Unable to Log In

If you're having trouble logging in, please refer to the guidance available through the "Can't log in?" button located at the bottom of the login screen.
